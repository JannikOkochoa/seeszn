# /public/briefs/

Place the exported PDF here as:

  ki-sichtbarkeits-brief-2026.pdf

To generate the PDF:
1. Open /brief/ki-sichtbarkeit/print in a browser
2. Print → Save as PDF
3. Page size: A4, Margins: None, Background graphics: enabled
4. Save as ki-sichtbarkeits-brief-2026.pdf and place in this directory

The download link on /brief/ki-sichtbarkeit points to /briefs/ki-sichtbarkeits-brief-2026.pdf
